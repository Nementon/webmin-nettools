Network Utilities - Whois
Whois is a tool to get information about registered domains.
Contents:
  • Domain name
  • Server

Domain name

 This is the name of the domain to be queried.

server

 The server to query. There are several servers hardcoded, but you may enter your own in the module configuration. It is a comma-separated list of server names. For example "whois.internic.net, whois.ripe.net".
