Network Utilities - IP Subnet Calculator
With this tool you can calculate the netmask for a TCP/IP-network.

Contents:
  • Calculate netmask by Class and Bits
  • Calculate netmask by number of hosts

Calculate netmask by Class and Bits

 You Choose a Network Class as they are these ones:
  • A - Max of 2^24 hosts. Private one is 10.0.0.0
  • B - Max of 2^16 hosts. Normally not needed for private nets
  • C - Max of 256 hosts per net. Most common network class. Private: 192.168.x.x
Then you choose the number of subent bits and it gives you the netmask.

Calculate netmask by number of hosts

You give the calculator a number and it returns the smallest network available.
