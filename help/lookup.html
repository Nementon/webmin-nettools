Network Utilities - Lookup
Lookup is a tool for obtaining the IP address from a hostname and vice-versa.
Contents:
  • Query Type
  • Nameserver
  • Timeout

Query Type

 In the DNS-Database are several types of records. The most important one is the A-record, the Address-Record. This is used for getting the address for an hostname. Another important one is the MX-Record, the Mail-eXchanger-Record. It specifies which host handles the mail for a specific domain.

Nameserver

If you choose Default nslookup will use the default server for the system Webmin is installed on.

Timeout

 If nslookup will not get an answer for this amount of seconds it will abort the lookup.
